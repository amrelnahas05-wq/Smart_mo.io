Safety
First

Activation
Blade Motor Conditions
  • Path sensors confirm obstacle-free zone
  • Proximity threshold verified (>30cm)
  • Emergency stop signal inactive
  • Battery voltage within safe range
Shutdown
Auto Shutdown Triggers
  • Obstacle detected within 15cm
  • Tilt / rollover sensor activated
  • ESP WiFi connection lost (>5 sec)
  • Battery critical level (<10%)
Control
Mobile Command Set
  • Forward — Accelerate drive motors
  • Backward — Reverse drive motors
  • Spin — Rotate in place (L/R)
  • Stop — Emergency halt + blade shutdown
